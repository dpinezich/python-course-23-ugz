# Cheatsheet

In diesem Cheatsheet finden Sie alle wichtigen Informationen und Shortcuts die im Kurs besprochen wurden. Dieses Cheatsheet wird auch ständig erweitert um so auf dem aktuellsten Stand zu bleiben. 

## Git

Git verwenden wir um das Kursmaterial einfach von Kursleiter zu Kursteilnehmer zu synchronisieren.
Die wichtigsten Kommandos hierzu sind 

`git clone` 

Um ein Repository (Speicher) auszuchecken. Dies wird nur einmal ganz am Anfang des Kurses notwendig sein.

`git pull`

Um die neusten Änderungen seitens Kursleitung zu erhalten und das eigene Repo auf den neusten Stand zu bringen.

`git stash`

Wenn aus versehen etwas ausserhalb des `userfolders` verändert wurde, kann es sein das ein `merge error` auftritt.
Diesen behebt man am schnellsten indem man die Änderungen "wegpackt" oder "stashed". Dies bedeutet das die Kursunterlagen (ausserhalb des `userfolders`)
wieder in den ursprünglichen Zustand versetzt werden und damit durch die Kursleitung überschrieben werden können. Will man eine Änderung behalten (ausserhalb des `userfolders`)
muss man die entsprechende Datei entweder in den `userfolder` kopieren oder sich kurz mit dem Kursleiter absprechen für andere Varianten.
